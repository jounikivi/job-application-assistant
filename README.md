# Job Application Assistant

Paikallisesti ajettava Python-työkalu, joka auttaa sovittamaan CV:n ja työpaikkailmoituksen yhteen.

Projektikansio sisältää tässä vaiheessa:
- rungon tulevaa toteutusta varten
- kattavan toteutussuunnitelman Markdown-muodossa
- PDF-version samasta suunnitelmasta
- apuskriptin PDF:n uudelleengenerointiin

Keskeinen tavoite:
- lukea työpaikkailmoituksen teksti
- verrata sitä CV:hen
- tunnistaa osuvat avainsanat ja osaamiset
- ehdottaa, mitä hakemuksessa kannattaa painottaa
- tuottaa luonnos osiolle "vahvuudet tähän rooliin"

Suunnitelmadokumentit löytyvät kansiosta `docs`.
